Wusic

A HUMAN TOUCH IN MUSICIANS' WORLD


Wusic is a platform helping musicians to find suitable partners for a project, or reverse. There are a lot of amazing musicians worldwide. But, there is a limited community for them to play around or create some awesome thing together. The goal of Wusic is to help musicians to find their soulmates.


Wusic is a Full-Stack Single-Page Web application, it's done with MERN Stack technology (MongoDB, Express, React & Node.js), it's developed during the last module of Ironhack Bootcamp, and it was awarded as the â€œHonorable Mention".

Experience

Wusic is the last project I've done during the intense Ironhack Bootcamp, the idea of Wusic came from a conversation between me and a friend of mine who is a musician, he explained to me the problem he's facing in making music, Since his music style is not mainstream, it is not easy for him to find like-minded music partners, so my purpose was aiming to create an app could help musicians to find musical partners which could match their vibes.


At that moment I was very bored of an ordinary design, so though we only had 1.5 weeks before the submission, I still spent 2~3 days on UX/UI design with Figma, in order to create a unique user experience.



User stories:

  • Signup: As an anon, the user can sign up to start the journey
  • Login: As a registered user, log in to visit his/her dashboard
  • Dashboard page: Shows the user's ongoing projects and let the user choose his/her partners, this page also shows the projects the user recently participated and his/her favorite musicians
  • Project searching page: Displays all the projects published by other users, you can search them with filters (project type, artist type and instrument ), and look for the interesting ones
  • Project detail page: Showing the detailed information of a project, here you can send a request to join the projects
  • Create project page: User can create a new project
  • Edit/delete project page: User can edit or delete his/her project
  • Musician searching page: Shows the musicians of Wusic platform, the user can search with filters (artist type and instrument)
  • Profile page: Showing all user info: basic info, type, description and Spotify embed
  • Edit/delete profile page: User can edit or delete his/her profile
  • About page: About Wusic
