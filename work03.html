CosLib

A BEAUTY PRODUCT LIBRARY


I created this website for those people who are interested in beauty products, the products can be filtered by types and brands read the description about the products and find the right one for you.


CosLib is developed with the HTML, CSS and JavaScript ES6 technologies, it's a very simple app, but it solidified my knowledge of OOP (Object Oriented Programming)


Tools:

  • HTML5
  • CSS3
  • Javascript (ES6)
  • OOP

Experience

I made CosLib during the first module of Ironhack, most of the content of the first module for me was as easy as pie, but still there are some points deserve to be recorded:


Remote working - To adapt the new normality, we applied the hybrid class mode, the bootcamp was being held with Zoom, it developed my telecommuting skills.


OOP - Object-oriented programming (OOP) is a computer programming model that organizes software design around data, or objects, rather than functions and logic. Due to the time constraints and to match the level of the students, our teacher only very superficially mentioned the OOP concept, however I'm aware of its importance. Finally could have a chance to practice it, and I'm very glad with my result!


External API - I applied the Makeup API with Fetch


Refactoring for compatibility - I developed CosLib with the advanced ES6 language, and one of the problem of the ES6 was the compatibility, in order to solve this problem, after the submission I complied CosLib with the help of Babel.
